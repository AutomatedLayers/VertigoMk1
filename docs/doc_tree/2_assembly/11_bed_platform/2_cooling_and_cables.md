## Assembly Notes

 - **7:11** These all get the same power and timing, so the order in which they enter the WAGOs is not important; just make sure to leave open the leftmost terminal in this animation frame.
 - **7:25** Extrusion slots are sometimes inconsistent across manufacturers. If the Flow Blocker Clips (or any of the other extrusion clips for that matter) are a bit loose, try scaling them in the axis that would cross the slot width ~1%.
