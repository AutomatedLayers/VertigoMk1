# Enclosure


#### Door

The door is actually a different project. You may want to design your own, or even simply use another acrylic panel and clips on the front since the Vertigo ejects parts out the bottom. It's nice to have a door, so one is provided in the kit, but ideally front entry would only be needed for occasional servicing. Check out the Clicky Clacky Door Assembly and GitHub below.

[Clicky Clacky Door Assembly](https://github.com/tanaes/whopping_Voron_mods/blob/main/clickyclacky_door%2FREADME.md#assembly)
